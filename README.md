## Abstract
The Laundry Management Application is a web-based dashboard designed for laundry service providers to manage their operations efficiently.  
It allows owners to:
- Track order status  
- Manage inventory  
- View reports on orders received, completed, and pending  
- Add or remove laundry services  
- Manage prices and promotions  
- Handle customer database and transaction history  
- Provide customer support  

The application increases efficiency and accuracy, reducing time and effort required for laundry management.

---

## Introduction
The project **"Laundry Management System"** is developed for laundry shops to manage customers and billing professionally.  

Key features:
- Professional billing and invoicing for personalized invoices  
- Tracking paid and unpaid invoices  
- Sending payment reminders to customers directly from the app  

Using this software, laundry shops can stand out with appealing invoices and better customer management.

## Methodology

### Modules:
1. **Creating Customer Database** – Register new customers with details (Name, Mobile No., Address)  
2. **Customer** – Stores customer details including previous unpaid bills  
3. **Services** – Owner can select services for customers (Iron, Only Wash, Wash + Iron)  
4. **Bill Generation** – Generates bills automatically  
5. **Bill Reminder** – Sends automatic reminders for unpaid bills
